与其它包一样, 安装`Polars`非常简单

[官方文档](https://docs.pola.rs/user-guide/installation/)

## 默认安装

### 使用pip安装

```shell
pip install polars
```
### 使用uv安装

```shell
uv add polars
```

## 额外tag
> 处理csv,excel等文件时需要额外的包, 例如`xlsx2csv`, `excel`等, 可以单独安装也可以在安装polars时一并安装
> 比如: `uv add polars[excel]`

```shell
pip install 'polars[tag]'
```

```shell
uv add 'polars[tag1,tag2,tag3]'
# []: 以逗号分隔的数组
```
