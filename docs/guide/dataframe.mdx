# DateFrame

`DataFrame`是polars的核心数据结构, 可以将它理解为一个n行m列组成的二维表格

```python
import polars as pl

# 下面就是一个DataFrame, dict中的三个key: "a","b","c" 就是三个列的列名,
# 它们的value分别是对应列的值
df: pl.DataFrame = pl.DataFrame(
      {
          "a": ["a", "b", "a", "b", "c"],
          "b": [1, 2, 1, 3, 3],
          "c": [5, 4, 3, 2, 1],
      }
)

print(df)
"""
下面是输出: 形状(5行3列)+数据, 同时注意到表格的表头也标注了每个列的数据类型
polars会自动解析数据类型, 关于类型解析稍后还会提到
tip: 如果有一列的长度和其他列不同, polars会报错
shape: (5, 3)
┌─────┬─────┬─────┐
│ a   ┆ b   ┆ c   │
│ --- ┆ --- ┆ --- │
│ str ┆ i64 ┆ i64 │
╞═════╪═════╪═════╡
│ a   ┆ 1   ┆ 5   │
│ b   ┆ 2   ┆ 4   │
│ a   ┆ 1   ┆ 3   │
│ b   ┆ 3   ┆ 2   │
│ c   ┆ 3   ┆ 1   │
└─────┴─────┴─────┘
"""
```
